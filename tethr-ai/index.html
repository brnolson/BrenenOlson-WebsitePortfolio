3 Where It'll Live

The new AI logo will replace the generic chat-bubble icon you see on the main chat screen (highlighted area). It'll appear:

  • As the chat avatar next to messages
  • As a loading indicator while the AI is "thinking"
  • Display animation to the user while it's generating new seating charts
↑ The purple chat-bubble icon next to the welcome message is what we're replacing.
The surrounding UI is dark/purple, so the new mark should look great on a dark background.

4 Logo Requirements

Create an AI-inspired version of the Tethr logo.

Style direction

  • Heart-shaped or heart-inspired (based on the current Tethr mark)
  • Purples + blues (Copilot / Gemini-style gradients)
  • Soft, modern, and magical
  • Should feel "AI-powered" without being overly techy

Inspiration ideas (optional)

Liquified / water-ripple Pencil drawn strokes and colors Holographic gradient Smooth dimensional transitions

File formats

  • SVG (important for scaling + UI use)
  • PNG (transparent background)
  • Any source file you're comfortable sharing (Figma, Illustrator, etc.)

5 Animation Requirements

A simple looping animation based on the AI logo.

Two small animations (if possible)

  • Idle loop — gentle ripple, drawn squiggle lines (like charlie brown), etc
  • Loading / working animation — slightly more active motion. Could be a ripple expanding, a morphing loop, or a soft "processing" effect.

File formats (whichever is easiest)

  • MP4 (cleanest for UI)
  • GIF (for previews)
  • Optional: PNG sequence if that's easier than exporting a GIF/MP4 — gives me flexibility to convert later
No pressure — if animation ends up being too much, I can try my hand at it another time lol.

6 Deliverables Summary

  • 1 final AI logo — SVG, PNGs, source file (optional)
  • 1–2 looping animations — MP4 or GIF, optional PNG frame sequence
